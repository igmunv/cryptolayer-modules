# CryptoLayer Modules

Чтобы ваш модуль был добавлен в этот репозиторий, отправьте PR с описанием, исходным кодом модуля и файлами: `requerements.txt`, `README.md`.

## Модули

- [VK](https://github.com/igmunv/cryptolayer-modules/tree/main/VK) - для российской социальной сети ВКонтакте. Авторизация происходит с помощью аккаунта пользователя.
- [VK Group](https://github.com/igmunv/cryptolayer-modules/tree/main/VK_group) - для российской социальной сети ВКонтакте. Авторизация происходит с помощью группы. Более стабильный и стойкий вариант.
- [Test](https://github.com/igmunv/cryptolayer-modules/tree/main/Test) - для тестирования работы CryptoLayer

---

To have your module added to this repository, submit a PR containing a description, the module's source code, and the required files: `requirements.txt` and `README.md`.

## Modules

- [VK](https://github.com/igmunv/cryptolayer-modules/tree/main/VK) - for the Russian social network VKontakte. Authorization is performed using a user account.
- [VK Group](https://github.com/igmunv/cryptolayer-modules/tree/main/VK_group) - for the Russian social network VKontakte. Authorization is performed using a group. A more stable and reliable option.
- [Test](https://github.com/igmunv/cryptolayer-modules/tree/main/Test) - for testing CryptoLayer performance.
