# VK
 
 Модуль для российскиой социальной сети ВКонтакте. Авторизация происходит от имени пользователя с помощью токена авторизации, который можно получить на этом сайте:

https://vkhost.github.io/

Выберите `vk.com`, авторизуйтесь, и далее будет страница с ошибкой и это нормально. Токен находится в адресной строке в поле `access_token`.

Лучше использовать модуль [VK Group](https://github.com/igmunv/cryptolayer-modules/tree/main/VK_group), который вместо обычных аккаунтов пользователей позволяет использовать группы, от имени которых отправляются сообщения. Это позволяет избежать капчи и блокировки аккаунтов. 

---

Module for the Russian social network VKontakte. Authorization is performed on behalf of a user using an authorization token, which can be obtained at this website:

https://vkhost.github.io/

Select `vk.com`, log in, and you will then see an error page—this is normal. The token is located in the address bar in the `access_token` field.

It is better to use the [VK Group](https://github.com/igmunv/cryptolayer-modules/tree/main/VK_group) module, which allows you to use groups instead of regular user accounts to send messages from. This helps avoid captchas and account bans.
